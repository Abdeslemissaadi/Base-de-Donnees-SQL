📊 Projet Base de Données SQL – Architecture Backend d’une Application Web
🚀 Présentation du projet

Ce projet consiste en la conception et la mise en œuvre d’une base de données relationnelle développée dans le cadre de la création d’une application web.

La base de données a été réalisée avec MySQL et administrée via phpMyAdmin.

L’objectif est de démontrer des compétences solides en :

modélisation de données
développement SQL avancé
gestion et structuration d’une base de données complexe
🧩 Fonctionnalités principales
✔️ Conception d’un schéma relationnel normalisé
✔️ Gestion de plusieurs tables interconnectées
✔️ Requêtes SQL avancées
✔️ Procédures stockées pour automatiser les traitements
✔️ Déclencheurs (triggers) pour garantir l’intégrité des données
✔️ Vues pour simplifier l’accès aux données
✔️ Fonctions SQL pour réutiliser la logique métier
🛠️ Technologies utilisées
MySQL
phpMyAdmin
SQL (DDL, DML)

🗄️ Structure de la base de données

La base est composée de plusieurs tables représentant la logique métier de l’application.
